# Install, Setup modules

```sh
$ brew install closure-compiler

$ git clone git@github.com:uupaa/DeviceQuery.js.git
$ cd DeviceQuery.js
$ npm install
```

# Minify

```sh
$ npm start

  or

$ node node_modules/uupaa.minify.js --keep --output ./lib/DeviceQuery.min.js ./lib/DeviceQuery.js
```

# Test

```sh
$ npm test
```
